# Dodge-the-Creeps
Este projeto é uma adaptação do tutorial oficial da documentação do Godot:  https://docs.godotengine.org/pt-br/4.x/getting_started/first_2d_game/index.html
## Sobre o projeto
Este jogo foi desenvolvido como forma de aprendizado utilizando a engine Godot 4.6.1

## Tecnologias utilizadas
- Godot Engine 4.6.1
- GDScript

## Objetivo
Praticar conceitos básicos de desenvolvimento de jogos, como:
- Movimentação
- Detecção de colisão
- Instanciação de cenas
- Organização de nós

## Créditos
Tutorial original disponível na documentação oficial do Godot.
